# 先进封装的组装方法

## 1. 专题范围

本页回答每种组装技术的三个问题：

1. **是什么**：该术语描述的是互连形式、组装顺序、键合对象还是键合机理。
2. **作用范围**：它适用于哪些芯片、载体和封装类型。
3. **主要作用**：它在封装结构中解决什么问题。

组装技术和封装类型不是同一维度。一个封装通常会组合多种技术，例如 2.5D 封装可以同时使用 `CoW + TCB + 微凸点倒装芯片`，封装完成后再通过 SMT 安装到 PCB。

## 2. 术语之间的关系

| 分类维度 | 回答的问题                               | 对应术语                |
| -------- | ---------------------------------------- | ----------------------- |
| 互连形式 | 芯片或器件以什么结构连接                 | SMT、引线键合、倒装芯片 |
| 键合对象 | 芯片与芯片、芯片与晶圆还是晶圆与晶圆连接 | CoC、CoW、WoW           |
| 键合机理 | 接合界面依靠什么方式形成                 | 回流焊、TCB、混合键合   |

这些术语并非同一维度的东西，可以叠加使用，例如：

- `CoC` 说明键合对象是两颗芯片，实际界面可以采用微凸点 TCB，也可以采用混合键合。
- `CoW` 说明把单颗芯片放到晶圆上，不限定使用焊料、Cu-Cu 还是混合键合。
- `Flip Chip` 说明芯片有源面朝向载体并通过面阵列端子连接，不限定采用批量回流还是 TCB。

## 3. 总索引

| 技术           | 具体是什么                                  | 主要作用                       | 主要关联封装类型                            |
| -------------- | ------------------------------------------- | ------------------------------ | ------------------------------------------- |
| SMT            | 将封装器件或表面贴装元件安装到 PCB/基板表面 | 完成板级或 SiP 系统集成        | 2D、2.1D、2.3D、2.5D、3D 封装的板级安装     |
| Wire Bond      | 用细金属线连接芯片焊盘和基板焊盘            | 引出外围 I/O，连接不同高度芯片 | [2D](IC集成/2D-IC集成.md)、[3D](IC集成/3D-IC集成.md) 封装 |
| Flip Chip      | 芯片有源面朝下，通过面阵列凸点连接载体      | 提高 I/O 密度并缩短互连        | 全部集成分类                                |
| CoC            | 单颗芯片与单颗芯片键合                      | 灵活构建局部 3D 堆叠           | [3D](IC集成/3D-IC集成.md)                          |
| CoW            | 单颗芯片与完整晶圆键合                      | 兼顾 KGD 选择与晶圆级集成      | [2.5D](IC集成/2.5D-IC集成.md)、[3D](IC集成/3D-IC集成.md)  |
| WoW            | 两片完整晶圆整体键合                        | 提高同构堆叠的并行度           | [3D](IC集成/3D-IC集成.md)                          |
| TCB            | 通过受控热量和压力形成键合界面              | 支持细间距微凸点和低焊料量互连 | 2.1D、2.5D、3D                              |
| Hybrid Bonding | 同时形成介质-介质和 Cu-Cu 连接              | 实现无凸点、超细间距垂直互连   | 主要用于 [3D](IC集成/3D-IC集成.md)                 |

## 4. 按互连形式分类

### 4.1 SMT（表面贴装技术）

SMT 是把封装后的 IC、电阻、电容、电感等表面贴装器件，通过焊点安装到 PCB 或 SiP 基板表面的组装技术。它主要属于板级或模块级组装，不是裸芯片内部高密度互连技术。

步骤大概如下，我愿称之为自动化机器焊板子：

1. 锡膏印刷： 像丝网印刷一样，用机器把锡膏精准涂在电路板的焊盘上。
2. 贴片： 贴片机以极高的速度把像米粒甚至沙子一样小的元器件抓取并精准“贴”在锡膏上。
3. 回流焊接： 把整块板子送进一个像隧道一样的“回流焊炉”里，通过高温让全板的锡膏融化，一次性把成百上千个点全部焊好。

**作用范围与封装类型**

- 用于把 2D、2.1D、2.3D、2.5D 或 3D 封装成品安装到 PCB。
- 用于 SiP 内部，将封装器件、射频器件和无源元件安装到共同基板。
- 用于 PoP、模块和系统板上的封装间连接。
  
总结来说，BGA ball 和 PCB 形成焊点，或者焊装电阻、电容等表面贴装器件，都是SMT技术。

### 4.2 引线键合（Wire Bond）

引线键合使用金、铜、铝等细金属线，把正面朝上的芯片焊盘连接到封装基板、引线框架或另一颗芯片的焊盘。它属于点到点的外围互连，连接路径位于芯片表面上方。

**作用范围与封装类型**

- 在 [2D IC 集成](IC集成/2D-IC集成.md) 中连接并排芯片与共同封装基板。
- 在 [3D IC 封装](IC集成/3D-IC集成.md#1-3d-ic-封装) 中连接错位堆叠的存储器芯片。
- 用于面对面堆叠中较大底层芯片的外露外围焊盘。
- 用于 PoP 顶部存储器封装和部分低成本 SiP。
- 不适合作为 2.5D 高带宽芯片间互连的主要方式。

**主要作用**

- 把芯片外围 I/O 引出到封装基板。
- 跨接不同高度、不同尺寸或不同焊盘位置的芯片。
- 以成熟、低成本方式实现多芯片和芯片堆叠连接。

它的互连较长、寄生电感较大，I/O 密度也受外围焊盘和线弧空间限制，因此高速、高带宽应用通常改用倒装芯片或混合键合。

### 4.3 倒装芯片（Flip Chip）

倒装芯片是把芯片有源面和焊盘朝向下方载体，通过 C4、C2 铜柱、微凸点或其他面阵列端子进行连接。与引线键合不同，I/O 可以分布在整个芯片表面，而不只位于芯片边缘。

**作用范围与封装类型**

| 封装类型               | 倒装芯片连接的位置                              |
| ---------------------- | ----------------------------------------------- |
| [2D](IC集成/2D-IC集成.md)     | 芯片直接连接封装基板或预制 Fan-out RDL          |
| [2.1D](IC集成/2.1D-IC集成.md) | 芯片连接基板顶部薄膜层或局部桥接区域            |
| [2.3D](IC集成/2.3D-IC集成.md) | 芯片连接无核心有机或 Fan-out RDL 转接板         |
| [2.5D](IC集成/2.5D-IC集成.md) | 芯片连接无源 TSV 硅转接板                       |
| [3D](IC集成/3D-IC集成.md)     | 上层芯片连接下层芯片、有源基底芯片或 HBM 相邻层 |

**主要作用**

- 提供高于引线键合的 I/O 密度。
- 缩短信号、电源和地的互连路径，降低寄生参数。
- 支持面阵列供电和高速 die-to-die 接口。
- 将芯片固定在基板、桥、转接板或另一颗芯片上。

## 5. 按键合对象分类

CoC、CoW 和 WoW 只说明“谁和谁键合”，不说明具体采用哪一种接合机理。三者都可以根据产品需要与 TCB、微凸点或混合键合组合。

### 5.1 CoC（Chip-on-Chip，芯片对芯片）

CoC 是把一颗已经切割的芯片直接键合到另一颗芯片上，形成单个局部芯片堆叠。

**作用范围与封装类型**

- 用于 [基于 TSV 的 3D CoC](IC集成/3D-IC集成.md#23-基于-tsv-的-coc)。
- 用于[带 TSV 的混合键合 CoC](IC集成/3D-IC集成.md#24-带-tsv-的无凸点混合键合-coc)。
- 用于[不带 TSV 的混合键合 CoC](IC集成/3D-IC集成.md#25-不带-tsv-的无凸点混合键合-coc)。
- 适合不同尺寸、不同工艺节点和不同功能的异构芯片。

**主要作用**

- 灵活选择两颗 KGD，降低坏芯片进入堆叠的风险。
- 构建逻辑-逻辑、逻辑-存储器、传感器-处理器等局部 3D 集成。
- 支持芯片尺寸不一致或仅局部需要垂直互连的设计。

### 5.2 CoW（Chip-on-Wafer，芯片对晶圆）

CoW 是把一颗颗已切割芯片键合到一片尚未切割的目标晶圆上。目标晶圆可以是器件晶圆、硅转接板晶圆或有源基底晶圆。

**作用范围与封装类型**

- 用于把 KGD 安装到 [2.5D 无源硅转接板](IC集成/2.5D-IC集成.md)。
- 用于逻辑芯片、存储器或芯粒与 3D 有源基底晶圆集成。
- 用于芯片到晶圆级混合键合。
- 适合异构芯片、不同芯片尺寸和高价值芯粒。

**主要作用**

- 同时利用单颗 KGD 选择能力和晶圆级批量处理能力。
- 允许不同来源、不同尺寸和不同工艺节点的芯片集成到同一晶圆载体。
- 在 2.5D/3D 集成中连接芯片与硅转接板或有源基底。

### 5.3 WoW（Wafer-on-Wafer，晶圆对晶圆）

WoW 是把两片完整晶圆整体对准并键合，使上下晶圆中对应的芯片同时形成连接。

**作用范围与封装类型**

- 主要用于 [3D IC 集成](IC集成/3D-IC集成.md)。
- 适合上下芯片尺寸、阵列位置和晶圆直径相容的同构或规则异构堆叠。
- 常见于图像传感器、存储器和规则阵列器件。
- 可与 Cu-Cu 或混合键合结合。

**主要作用**

- 一次完成整片晶圆上的大量芯片连接，提高组装并行度。
- 提供统一的晶圆级对准和后续减薄、背面互连平台。
- 适合结构重复、产量大且上下晶圆良率较高的产品。

WoW 的整体利用率受两片晶圆良率和芯片阵列匹配影响，因此它不如 CoC/CoW 灵活。

### 5.4 CoC、CoW 与 WoW 的适用差异

| 维度           | CoC                | CoW                   | WoW                |
| -------------- | ------------------ | --------------------- | ------------------ |
| 键合对象       | 单颗芯片对单颗芯片 | 单颗芯片对完整晶圆    | 完整晶圆对完整晶圆 |
| KGD 选择能力   | 高                 | 高                    | 依赖晶圆图谱匹配   |
| 芯片尺寸兼容性 | 高                 | 高                    | 较低               |
| 组装并行度     | 低                 | 中                    | 高                 |
| 主要封装范围   | 局部 3D 堆叠       | 2.5D、3D              | 3D                 |
| 核心作用       | 灵活构建堆叠       | 异构 KGD 的晶圆级集成 | 大批量规则堆叠     |

## 6. 按键合机理分类

### 6.1 回流焊键合

回流焊键合利用焊料熔化和润湿，使 C4、C2 或焊球与对应焊盘形成连接。它可以同时连接同一载体上的多颗芯片或大量凸点。

**主要作用**

- 以成熟方式批量形成焊料互连。
- 利用焊料自对准能力吸收一定贴装偏差。
- 提供电气连接以及一定的机械支撑和应变顺应性。

### 6.2 TCB（热压键合）

TCB 是在键合界面施加受控热量和压力，使微凸点焊料、Cu-Cu 或其他金属界面形成连接。它和回流焊的主要差别是对单颗芯片施加局部压力，因此能更主动地控制细间距界面的接触和共面性。

**主要作用**

- 支持比普通批量回流更细的凸点间距。
- 通过压力改善翘曲芯片和低焊料量凸点的接触。
- 减少细间距焊料桥连，并控制堆叠后的界面高度。
- 为 2.5D 和微凸点 3D 集成提供高密度连接。

### 6.3 混合键合（Hybrid Bonding）

混合键合是在同一平面上同时形成介质-介质机械连接和 Cu-Cu 电气连接。铜焊盘嵌入平坦化介质表面，界面不使用传统焊料微凸点，因此也称**无凸点键合**。

**主要作用**

- 去除焊料微凸点及其高度，缩短垂直互连。
- 支持比微凸点更细的连接间距和更高 I/O 密度。
- 降低互连电阻、电感和寄生电容，改善带宽与能效。
- 减薄芯片堆叠并减少传统底填间隙。

它的适用前提是键合表面具有很高的平坦度、洁净度和对准精度，且界面通常不可返修。

### 6.4 微凸点与混合键合的功能差异

| 对比项               | 微凸点 + TCB/回流        | 混合键合                       |
| -------------------- | ------------------------ | ------------------------------ |
| 界面结构             | 铜柱/焊料微凸点          | 介质-介质 + Cu-Cu              |
| 是否有凸点高度       | 有                       | 无传统凸点                     |
| 是否需要填充界面间隙 | 通常需要                 | 通常无独立底填间隙             |
| 主要封装范围         | 2.1D、2.5D、3D           | 主要为 3D                      |
| 核心作用             | 成熟地实现细间距芯片连接 | 实现超细间距、低寄生垂直互连   |
| 主要限制             | 焊料桥连、IMC 和间距缩放 | 表面质量、颗粒、对准和不可返修 |
