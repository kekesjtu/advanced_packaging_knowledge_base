## 9. Underfill（底部填充）

### 9.1 作用与基本组成

Underfill 是填充在芯片与基板、转接板或相邻芯片之间的聚合物材料。它包覆焊点或微凸点，把局部载荷分散到更大的界面区域，降低芯片、焊点与基板因热膨胀系数不同而产生的应变，同时提高抗跌落、抗振动、抗湿气和温度循环能力。

常见材料以环氧树脂为基体，并包含二氧化硅填料、固化剂、增韧剂、偶联剂和流动控制组分。材料与工艺需要同时匹配凸点间距、间隙高度、芯片尺寸、基板表面、回流或热压温度以及后续可靠性要求。

### 9.2 毛细底填 CUF

毛细底填（Capillary Underfill，CUF）在芯片贴装和焊点回流完成后进行。材料从芯片一侧或多侧点胶，依靠毛细作用流入芯片底部，随后加热固化。

```txt
芯片贴装与回流
→ 助焊剂清洗或表面预处理
→ 预热
→ 边缘点胶
→ 毛细流动填充
→ 固化
```

CUF 工艺成熟，材料选择多，适用于倒装芯片、BGA、CSP 和部分微凸点封装。其主要限制是填充时间随芯片面积增大和间隙减小而增加，并容易出现流动前沿包气、填充不完整、填料分离和焊点周围空洞。点胶位置、基板温度、材料黏度、润湿性、填料粒径和芯片底部洁净度是关键控制项。

### 9.3 无流动底填 NUF

无流动底填（No-flow Underfill，NUF）在芯片贴装前点涂到基板上。芯片放置后，材料在回流过程中完成助焊、焊点形成和预固化，之后再进行最终固化。

NUF 把回流与底填结合，可减少回流后的毛细填充工序，适合较小芯片和对产能敏感的封装。其工艺窗口较窄：材料必须允许焊点在固化前充分塌陷和润湿，且不能因填料、挥发物或过早凝胶化阻碍焊接。常见风险包括焊点开路、填料夹入焊接界面、树脂空洞和芯片漂移。

### 9.4 模塑底填 MUF

模塑底填（Molded Underfill，MUF）使用模塑料在一次模塑过程中同时填充芯片底部间隙并包封芯片。它省去独立 CUF 点胶和固化步骤，适合高产量倒装封装、扇出封装和多芯片封装。

MUF 对模塑料流动性、填料粒径、模压条件和排气设计要求较高。细间距、小间隙或大面积芯片容易发生底部填充不完整、空洞、填料堵塞、芯片偏移和封装翘曲。模塑压力还需与薄芯片、微凸点和脆弱 RDL 的承载能力匹配。

### 9.5 晶圆级底填 WLUF

晶圆级底填（Wafer-level Underfill，WLUF）在晶圆切割前，以液态涂布或薄膜贴合方式把底填材料预先形成在芯片表面。晶圆切割后，带有底填材料的芯片再进行贴装和键合。

WLUF 可把单颗芯片点胶转为晶圆级批量处理，适合高密度和高产量组装。材料必须兼容晶圆切割、芯片拾取、对准、焊接或热压键合，并避免污染焊盘。膜厚均匀性、切割边缘质量、材料储存稳定性、键合时的树脂流动和排气是主要控制项。

### 9.6 常用工艺对比

| 类型 | 材料施加时机 | 主要优点                           | 主要限制                                   |
| ---- | ------------ | ---------------------------------- | ------------------------------------------ |
| CUF  | 贴装和回流后 | 工艺成熟、材料选择多、焊点可先检查 | 增加点胶与固化工序，大芯片和小间隙填充较慢 |
| NUF  | 贴装前点胶   | 回流与底填结合，流程较短           | 焊接与固化窗口耦合，材料和回流控制要求高   |
| MUF  | 模塑时       | 同时完成底填和包封，适合批量生产   | 小间隙填充、模压载荷和翘曲控制较难         |
| WLUF | 晶圆切割前   | 晶圆级批量施加，单颗组装步骤少     | 需兼容切割、拾取、对准和键合全过程         |
