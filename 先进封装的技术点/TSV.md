## 5. TSV

### 5.1 TSV 类型与应用

TSV 是贯穿硅衬底的垂直导电通道。在 [2.5D](../IC集成/2.5D-IC集成.md) 中，TSV 位于无源硅转接板内，负责把顶部 RDL 的信号和电源引到底部封装基板；在 [3D](../IC集成/3D-IC集成.md) 中，TSV 可位于逻辑、存储器或有源基底芯片中，实现芯片堆叠的垂直互连。

按填充方式可分为实心铜 TSV、环形 TSV 等；按结构可分为盲孔制程后减薄露头和直接通孔。几何设计需同时考虑直径、深宽比、间距、寄生参数、载流能力和硅中应力影响。

### 5.2 Via-first、Via-middle 与 Via-last

| 集成时机   | 制作阶段                     | 主要特点                                                                   |
| ---------- | ---------------------------- | -------------------------------------------------------------------------- |
| Via-first  | 晶体管制造之前               | 可承受较高后续温度，但 TSV 会占用器件布局区域并受前段洁净要求约束          |
| Via-middle | 晶体管完成后、顶层互连完成前 | 兼顾互连接近性和制程整合，需限制热预算                                     |
| Via-last   | 晶圆互连完成后               | 与既有芯片流程解耦；可从正面或背面加工，但深宽比、对准和薄晶圆处理更具挑战 |

选择不只取决于成本，还取决于代工流程所有权、允许温度、TSV 与有源区距离以及是否需要背面供电或背面 RDL。

### 5.3 深硅刻蚀、绝缘、填充、减薄与露头

典型铜 TSV 流程为：

```txt
光刻定义 TSV
→ DRIE 深硅刻蚀
→ 侧壁绝缘层
→ 阻挡层和铜种子层
→ 铜电镀填充
→ CMP 去除表面过镀铜
→ 临时键合到载片
→ 背面减薄
→ TSV 露头、钝化和背面 RDL / UBM
→ 解键合
```

关键风险包括 DRIE 侧壁扇贝和底部缺口、绝缘层针孔、种子层断裂、填铜空洞、CMP 凹陷、减薄破片、铜泵出，以及铜/硅 CTE 不匹配形成的应力和器件禁布区。薄晶圆在后续键合、清洗和搬运中必须依赖稳定的临时载片系统。
