## 4. 重布线层（RDL）

### 4.1 RDL 结构与功能

RDL 通过“介质层 + 铜走线 + 铜过孔”的重复堆叠，把芯片上细间距焊盘重新分配到凸点、焊球或相邻芯片。它承担三类功能：

- I/O 扇入或扇出。
- 芯片间高密度横向互连。
- 电源/地分配、阻抗控制和封装级无源结构。

主要设计参数是线宽/线距、铜厚、过孔直径与间距、层数、焊盘/凸点间距、走线长度、电流密度和介质厚度。缩小线宽/线距可提高布线密度，但会收紧光刻、对位、残铜、开路/短路和电迁移的工艺窗口。

典型加成法流程为：

```txt
介质涂布、固化与开孔
→ 溅射阻挡层/铜种子层
→ 光刻形成电镀图形
→ 铜电镀
→ 去胶
→ 蚀刻暴露的种子层
→ 重复堆叠并形成终端焊盘
```

### 4.2 导体与介质材料

导体：

- 主导体：Cu
- 粘附/阻挡层：Ti、TiW、Ta、TaN、Cr、Ni 等：这些层很薄，让铜和介质粘得住，并阻止Cu向介质中扩散。

| RDL 类型   | 常见介质                      | 特点与适用位置                                                                                                    |
| ---------- | ----------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| 有机 RDL   | PI、PBO、BCB 等聚合物         | 成本较低，适合大面积/面板级和多层扇出封装；线宽线距通常不如无机RDL细；需控制吸湿、固化收缩和 CTE 不匹配；热阻较大 |
| 无机 RDL   | SiO2、SiNx 等                 | 设备和成本接近晶圆制造（贵），可支持更细线宽/线距； 不像有机聚合物那样能缓冲热应力（脆）                          |
| Hybrid RDL | 芯片侧无机细线 + 外围有机粗线 | 兼顾超细间距和较大扇出面积，但增加异质界面和制程复杂度                                                            |
