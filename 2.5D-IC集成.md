# 2.5D-IC 集成

2.5D的共同特征是：芯片由无源TSV（硅通孔）中介层支撑，该中介层再贴装到封装基板上。与带有 CMOS 器件的有源TSV中介层（在3D集成中讨论）不同，无源TSV中介层只是一块带有 TSV 和 RDL 的裸硅片。

## 1. 基于无源 TSV 转接板的 2.5D IC 集成

```txt
空气 / 塑封料 / lid + TIM + 可选散热片
→ 多颗 silicon die / chiplet / HBM 的背面，器件并排且有源面朝下
→ microbumps + underfill
→ 无源 silicon interposer 顶部 fine-pitch RDL
→ passive silicon interposer（含 TSV，但不含 CMOS 有源器件）
→ interposer 背面 RDL / UBM + C4 bumps
→ build-up package substrate
→ BGA solder balls
→ PCB
```

`microbumps + 顶部 RDL` 负责高密度芯片间横向互连，`TSV + C4 bumps` 负责把信号、电源和地垂直引到封装基板。

### 1.1 PDF 中的具体商业产品

1. Xilinx / TSMC `Virtex-7 HT` FPGA：2013 年宣布量产，PDF 称其为业界首个进入生产的 2.5D IC 集成；4 颗 28 nm FPGA die 通过 CoWoS 硅转接板互连（pp.322–325）。
2. AMD `Radeon R9 Fury X` / `Fiji` GPU：2015 年出货；TSMC 28 nm GPU + 4 颗 SK hynix HBM，UMC 制造 `28 mm × 35 mm` 硅转接板，ASE 完成组装（pp.326–328）。
3. NVIDIA `Tesla P100` / `Pascal GP100`：2016 年出货；TSMC 16 nm GPU + 4 颗 Samsung HBM2（16 GB），采用 TSMC CoWoS 硅转接板（pp.328–329）。

### 1.2 PDF 中的厂商平台 / 商业方案

1. TSMC `CoWoS`（chip-on-wafer-on-substrate）：逻辑与存储器先安装到被动硅 TSV interposer，再连接封装基板；PDF 将其作为 2.5D 主流平台（pp.320–322）。
2. TSMC `CoWoS Roadmap`：从 2011 年约 `1× reticle` 扩展到规划中的 `3×/4× reticle`，用于容纳更大的处理器和更多 HBM（pp.329–330）。
3. TSMC `CoWoS + DTC`：在硅转接板中集成 deep trench capacitor，面向逻辑 + HBM2E 的 HPC 平台（pp.330–331）。
4. Xilinx / TSMC `CoWoS XL` 测试平台：3 颗 FPGA + 2 颗 HBM，使用 `31.5 mm × 41.7 mm` TSV interposer（pp.324–325）。
5. Altera / TSMC `CoWoS`：芯片通过微凸点连接 TSMC TSV interposer，再以 C4 连接 `6-2-6` 基板；PDF 明确说明该方案未进入 HVM（p.326）。

### 1.3 PDF 收录的研发、验证和特殊用途方案

以下项目不是都已商品化，但均是 PDF 第 6 章列出的 2.5D 实例，故完整保留并明确标为研发/验证方案。

1. Leti `SoW`（system-on-wafer）：ASIC、memory、PMIC、MEMS 与无源器件集成在带 TSV/RDL 的硅晶圆上（pp.299–300）。
2. IME `3D nonlinear local/global analysis`：面向 `21 mm × 21 mm`、11,000 I/O 大芯片的 TSV interposer 热机械验证（pp.300–303）。
3. IME `Electrical and Fluidic Interconnects`：双层 TSV interposer 同时提供电互连和液冷微通道（pp.303–305）。
4. IME `Double-Stacked Passive TSV Interposers`：两层转接板承载 RF、logic 和堆叠 memory（pp.305–306）。
5. IME `TSV Interposer as Stress Buffer`：利用硅转接板与专用 underfill 降低大芯片 Cu/low-k 焊盘应力（pp.306–308）。
6. HKUST `TSV Interposer with Chips on Both Sides`：高功率芯片在顶面、低功率芯片在底面，并配合 heat spreader（p.308）。
7. ITRI `Thermal Management with Chips on Both Sides`：双面芯片 + Cu heat slug 的热管理方案（pp.308–311）。
8. ITRI `Embedded Fluidic Microchannels for LEDs`：面向 LED/ASIC 的转接板内嵌液冷通道（pp.311–313）。
9. ITRI `SoCs and Memory Cube`：memory cube、thermal/mechanical chip、RDL、TSV 和 IPD 的测试载具（pp.313–315）。
10. ITRI `Semi-Embedded TSV Interposer`：转接板半嵌入有机基板，并保留 stress-relief gap（p.315）。
11. ITRI `Double-Sided Chip Attachments`：顶面 CPU/logic、底面双 memory 的双面贴装方案（pp.315–317）。
12. ITRI `TSV Interposer with Chips on Both Sides`：顶面两颗芯片、底面一颗芯片的另一种双面结构（pp.317–318）。
13. ITRI `TSH Interposer`（through-silicon-hole interposer）：硅孔不做侧壁金属化/填铜，以 Cu pillar 穿孔连接上下芯片（pp.318–322）。
14. IME `Non-destructive Fault Isolation`：用于定位 2.5D interposer/基板金属开路的 PFA 方法（pp.331–332）。
15. Fraunhofer `Photonics TSV Interposer`：集成 PD、TIA、driver、VCSEL、光/电 TSV，并连接玻璃 OPCB（pp.332–333）。
16. Dai Nippon Printing / AGC `Glass Interposer for AiP`：石英/玻璃转接板、TGV 与 `2 μm` Cu 布线，面向高频高速天线封装（pp.333–335）。
17. Fujitsu `Multilayer Glass Interposer`：LIDE 制作 TGV、导电膏填孔，并以 `40 μm` pitch microbump 连接芯片（pp.334–335）。
