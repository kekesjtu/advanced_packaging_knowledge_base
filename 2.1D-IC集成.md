# 2.1D-IC 集成

2.1D 的共同特征是：高密度互连只位于积层基板顶部或局部桥接区域，不使用覆盖整个封装的大面积独立转接板。

## 1. 积层封装基板顶部薄膜层

```txt
空气 / 塑封料 / 可选 lid、TIM 与散热片
→ 多颗 silicon die 的背面，芯片并排且正面朝下
→ microbumps / C2 bumps + underfill
→ 基板顶部 fine-pitch thin-film RDL
→ 常规 build-up layers
→ package substrate core + 背面 build-up layers
→ BGA solder balls
→ PCB
```

![Shinko i-THOP 顶部薄膜层](/pictures/2.1D/薄膜层-iTHOP.png)

图片来源：`[PDF]` p.251，Fig. 5.17。

商业方案 / PDF 收录方案：

1. Shinko `i-THOP`（integrated thin-film high-density organic package）：在常规积层基板顶部增加 2～4 层薄膜 Cu RDL，线宽/线距可达 `2/2 μm`，支持 `40 μm` pitch 微凸点（pp.251–253）。
2. Hitachi `2.1D Organic Interposer`：顶部高密度有机布线达到 `2/2 μm`，微孔直径 `5 μm`（pp.253–254）。
3. ASE `2.1D Organic Interposer`：基于 `512 mm × 408 mm` 大面板制造，每个 `23 mm × 23 mm` 单元承载两颗芯片（pp.254–256）。
4. SPIL `2.1D Organic Interposer`：`45 mm × 45 mm` 封装，4 层薄膜层，最小 `2/2 μm` L/S、`40 μm` 微焊盘间距（pp.254–258）。
5. JCET `uFOS`（ultra format organic substrate）：在无核心基板上制作 `2/2 μm` 薄膜布线，并加入嵌入式金属加强层 `e-STF` 控制翘曲（pp.255–259）。

## 2. 积层封装基板内嵌桥接结构

```txt
空气 / 塑封料 / 可选 lid、TIM 与散热片
→ 多颗 silicon die 的背面，芯片并排且正面朝下
→ 芯片间局部区域使用 microbumps；其余 I/O 使用 C2 / C4 bumps + underfill
→ build-up package substrate 顶层
→ 局部嵌入的 silicon bridge / high-density bridge（细间距 RDL 横向连接相邻芯片）
→ package substrate core 与其余 build-up layers
→ BGA solder balls
→ PCB
```

桥只覆盖需要高速 die-to-die 互连的局部区域；电源和低密度 I/O 仍可直接进入封装基板。

![Intel EMIB 与 Agilex FPGA](/pictures/2.1D/EMIB.png)

图片来源：`[PDF]` p.260，Fig. 5.28。

商业方案 / 具体产品：

1. Intel `EMIB`（embedded multi-die interconnect bridge）：局部硅桥嵌入积层基板顶层，以细间距 RDL 连接相邻芯片。
2. Intel `Agilex FPGA` 模块：PDF 给出的 EMIB 具体产品例子；FPGA 同时使用 C4 bump 和 C2 microbump，与周边芯片通过 EMIB 横向互连（pp.255–260）。

## 3. 扇出 EMC 内嵌桥接结构

```txt
空气 / 塑封料 / 可选散热结构
→ 多颗 face-down silicon die + EMC
→ 嵌于 EMC、位于相邻芯片之间或下方的局部 silicon bridge
→ fan-out RDL（连接芯片焊盘、桥上细间距 RDL 和外部焊盘）
→ TMV / TIV 等穿模垂直互连
→ BGA solder balls，或 C4 bumps + package substrate + BGA solder balls
→ PCB
```

![扇出 EMC 内嵌桥接结构](/pictures/2.1D/扇出桥接.png)

图片来源：`[PDF]` p.260，Fig. 5.29。

商业方案：

1. Applied Materials `Bridge`：将局部硅桥嵌入 EMC，芯片通过 fan-out RDL 与桥横向互连，垂直方向使用 TMV（pp.256–257）。
2. TSMC `LSI`（local silicon interconnect）：同属 EMC/RDL 中的局部桥接方案；PDF 指出 LSI 除 TSV-less 硅桥外，还可能包含 TSV 或 CMOS 器件（pp.257、261）。
