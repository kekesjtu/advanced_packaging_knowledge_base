# 先进封装中文知识库

面向先进封装技术学习、工程检索与知识整理的中文知识库。

本项目以集成形态为主线，以组装工艺和关键技术为横向专题，整理 2D、2.1D、2.3D、2.5D 与 3D IC 集成相关概念。内容定位为技术知识框架，不替代标准、论文、产品手册或制造商正式文档。

## 知识结构

### 按集成形态

| 分类 | 核心判据 | 文档 |
| --- | --- | --- |
| 2D IC 集成 | 多芯片在封装基板或重布线层上并排集成 | [2D-IC 集成](IC集成/2D-IC集成.md) |
| 2.1D IC 集成 | 积层基板顶部具有薄膜层，或在基板/EMC 中嵌入桥接结构 | [2.1D-IC 集成](IC集成/2.1D-IC集成.md) |
| 2.3D IC 集成 | 使用无核心、无 TSV 的有机或扇出型转接板 | [2.3D-IC 集成](IC集成/2.3D-IC集成.md) |
| 2.5D IC 集成 | 使用无源 TSV 硅转接板承载多芯片 | [2.5D-IC 集成](IC集成/2.5D-IC集成.md) |
| 3D IC 集成 | 芯片垂直堆叠，并采用 TSV、微凸点或无凸点互连 | [3D-IC 集成](IC集成/3D-IC集成.md) |

## 阅读方式

1. 根据封装结构进入相应的集成分类文档。
2. 通过文档内链接查阅组装方法、凸点或关键技术专题。
3. 对产品参数、标准版本和产业状态等时效性信息，回到权利人或标准组织发布的正式资料核验。

## 内容规范

- 分类文档只在其分类边界内组织内容，避免混入无关技术条目。
- 首次出现的缩略语应同时给出中文名称和英文全称。
- 技术事实应尽量注明资料类型、出处和页码；产品现状应注明信息对应的时间。
- `[PDF]` 表示对本地研究资料的溯源标记，仅用于定位出处，不是下载链接，也不表示该资料已获得公开传播授权。
- 引用内容应以概括、转述和独立分析为主，不大段复制第三方文字。
- 图片随知识库发布，但必须直接服务于具体技术的介绍、评论或说明，不作为装饰或原始资料的替代品。
- 第三方图片应紧邻相关正文，并标注作者或机构、资料名称、页码/图号及原始来源；许可状态明确时一并标注。
