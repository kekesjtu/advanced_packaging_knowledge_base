# 2.3D-IC 集成

2.3D 的共同特征是：多颗芯片先落在一个覆盖封装主要区域、但不含 TSV 的独立有机或无机 RDL 转接板上，该转接板再连接积层封装基板。

## 1. 积层封装基板上的无核心有机转接板

```txt
空气 / 塑封料 / 可选 lid、TIM 与散热片
→ 多颗 silicon die 的背面，芯片并排且正面朝下
→ microbumps / C2 bumps + underfill
→ coreless organic interposer（全部由积层布线和微孔组成，无厚芯层、无 TSV）
→ C4 bumps
→ build-up package substrate
→ BGA solder balls
→ PCB
```

![Cisco 有机转接板](/pictures/2.3D/Cisco有机转接板.png)

图片来源：`[PDF]` p.262，Fig. 5.32。

商业方案 / 具体产品：

1. `[PDF]` Sony PlayStation 3 `Cell` 处理器用 coreless substrate：PDF 将其列为 2010 年首个无核心基板量产例子（p.259）。
2. `[PDF]` Shinko `Coreless Organic Interposer`：以无核心有机基板代替 TSV interposer，可承载并排芯片或“逻辑芯片 + memory cube”（pp.260–262）。
3. `[PDF]` Cisco `Organic Interposer`：`38 mm × 30 mm × 0.4 mm`、10 层高密度有机转接板，承载一颗高性能 ASIC/FPGA 和 4 颗 HBM 堆叠（pp.262–263）。

## 2. 积层封装基板上的扇出 RDL 转接板（Chip-first 与 Chip-last）

两种流程的完成态拓扑基本相同，主要差别位于“芯片—RDL”界面：

```txt
空气 / 塑封料 / 可选 lid、TIM 与散热片
→ 多颗 silicon die + EMC，芯片并排且有源面朝向下方 RDL
→ Chip-first：RDL 直接制作并连接芯片焊盘，无独立 microbump 界面
  或 Chip-last：芯片通过 microbumps + underfill 连接预制 RDL
→ coreless fan-out RDL interposer（无 TSV）
→ C4 bumps
→ build-up package substrate
→ BGA solder balls
→ PCB
```

### 2.1 Chip-first 商业方案

![ASE FOCoS Chip-first](/pictures/2.3D/FOCoS-chip-first.png)

图片来源：`[PDF]` p.267，Figs. 5.39–5.40。

1. `[PDF]` STATS ChipPAC `2.3D eWLB / FOFC-eWLB`：以 chip-first fan-out RDL 取代 TSV interposer、microbump 和芯片侧 underfill（pp.263–265）。
2. `[PDF]` MediaTek `Fan-Out Chip-First Organic Interposer`：用 FOWLP 制作 TSV-less RDL interposer，并以 microbump 连接 `6-2-6` package substrate（pp.264–266）。
3. `[PDF]` ASE `FOCoS Chip-First`（fan-out wafer-level chip-on-substrate）：芯片先重构、塑封，再制作 RDL，底部通过 C4 bump 接封装基板（pp.266–268）。
4. `[PDF]` TSMC `InFO_oS`（Integrated Fan-Out on Substrate）：用于异构集成，以 fan-out RDL 取代 TSV interposer、microbump 和 underfill（pp.266–268）。
5. `[PDF]` TSMC `InFO_MS`（Integrated Fan-Out with Memory on Substrate）：面向更高性能的逻辑/存储集成（p.268）。

### 2.2 Chip-last / RDL-first 商业方案

![ASE FOCoS Chip-last](/pictures/2.3D/FOCoS-chip-last.png)

图片来源：`[PDF]` p.272，Fig. 5.48。

1. `[PDF]` SPIL `NTI`（non-TSV interposer）：用 65 nm 工艺制作最小 `0.4 μm` pitch 的无机 RDL interposer，芯片经 microbump/underfill 后贴装（pp.268–270）。
2. `[PDF]` Samsung `Si-Less RDL Interposer`：面向 HPC 的 `55 mm × 55 mm`、5 层 RDL 转接板方案（pp.269–272）。
3. `[PDF]` ASE `FOCoS Chip-Last`：先在临时玻璃载板上制作至少 4 层 RDL，再进行芯片键合、底填、塑封和 C4 bump 制作（pp.271–273）。
4. `[PDF]` TSMC `Multilayer RDL Interposer`：6 层 RDL，可采用 staggered vias、两层或四层 stacked vias（pp.273–274）。
5. `[PDF]` Shinko `2.3D Coreless Organic Interposer`：面向 HPC，芯片侧使用 SnAgCu，转接板与积层基板之间使用 SnBi + NCF（pp.273–276）。
6. `[PDF]` Unimicron `2.3D RDL-Interposer`：基于 `515 mm × 510 mm` 玻璃载板的 panel-level RDL-first 方案，验证一颗 `10 mm × 10 mm` 与一颗 `5 mm × 5 mm` 芯片的异构集成（pp.276–294）。
