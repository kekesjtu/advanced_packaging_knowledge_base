# 2.5D-IC 集成

2.5D 的共同特征是：多颗芯片由同一块无源 TSV/RDL 硅转接板支撑，芯片之间通过转接板顶层细间距 RDL 做横向互连，转接板再通过 TSV 和 C4 bumps 连接到积层封装基板。

与 2.1D 相比，2.5D 使用大面积独立硅转接板，而不是基板顶部薄膜层或局部硅桥。与 2.3D 相比，2.5D 的典型转接板含 TSV。与 3D IC 集成相比，2.5D 的主要芯片在封装层面并排放置；HBM 本身可以是 3D 堆叠，但 HBM 与 GPU/SoC 的关系仍是并排放在 interposer 上。

以下描述均按完成封装后的典型截面从上到下排列。

## 1. 基于无源 TSV 硅转接板的标准拓扑

```txt
塑封料 / heat spreader
→ 多颗 silicon die / chiplet / HBM stack，背面朝上，器件并排且有源面朝下
→ microbumps + underfill
→ passive silicon interposer + 顶部 fine-pitch RDL
→ C4 bumps + underfill
→ build-up package substrate
→ BGA solder balls
→ PCB
```

`microbumps + 顶部 RDL` 负责高密度、短距离、横向 die-to-die 互连；`TSV + C4 bumps` 负责把信号、电源和地垂直引到封装基板。

![Xilinx 与 TSMC 的 CoWoS FPGA](/images/2.5D/CoWoS-FPGA.png)

图片来源：`[PDF]` p.323，Figs. 6.30-6.31。

商业方案 / PDF 收录方案：

1. `[PDF]` Xilinx / TSMC `Virtex-7 HT` FPGA：PDF 称其为业界首个进入生产的 2.5D IC 集成；4 颗 28 nm FPGA die 通过 CoWoS 硅转接板互连，每颗 FPGA 有超过 50,000 个 `45 μm` pitch microbumps，转接板承载超过 200,000 个 microbumps（pp.322-325）。
2. `[PDF]` TSMC `CoWoS`（chip-on-wafer-on-substrate）：逻辑与存储器先安装到 passive silicon interposer，再连接封装基板；PDF 将其作为 2.5D 主流平台（pp.320-322）。
3. `[PDF]` Altera / TSMC `CoWoS`：芯片通过 Cu-pillar + solder cap microbumps 连接 TSMC TSV interposer，再以 C4 连接 `6-2-6` 封装基板；PDF 明确说明该方案未进入量产阶段（HVM）（p.326）。

## 2. GPU / SoC + HBM 的典型产品拓扑

```txt
塑封料 / heat spreader
→ 一颗大 GPU / SoC / accelerator die + 多颗 HBM stack，并排放在同一 interposer 上
→ microbumps + underfill
→ passive silicon interposer + 顶部 RDL（GPU/SoC 与 HBM 的超宽并行互连）
→ C4 bumps
→ build-up package substrate
→ BGA solder balls
→ PCB
```

这类拓扑是当前 2.5D 最典型的产品形态：中央或近中央放置大逻辑芯片，周边放置多颗 HBM stack。HBM stack 内部是 3D 存储器堆叠，但 GPU/SoC 与 HBM 在封装层面仍是 2.5D 横向集成。

![AMD Radeon R9 Fury X / Fiji](/images/2.5D/AMD-Fiji.png)

图片来源：`[PDF]` p.327，Fig. 6.36。

![NVIDIA P100](/images/2.5D/NVIDIA-P100.png)

图片来源：`[PDF]` p.328，Fig. 6.38。

商业方案 / 具体产品：

1. `[PDF]` AMD `Radeon R9 Fury X` / `Fiji` GPU：2015 年出货；TSMC 28 nm GPU + 4 颗 SK hynix HBM，UMC 制造 `28 mm × 35 mm` 硅转接板，ASE 完成组装（pp.326-328）。
2. `[PDF]` NVIDIA `Tesla P100` / `Pascal GP100`：2016 年出货；TSMC 16 nm GPU + 4 颗 Samsung HBM2，采用 TSMC CoWoS 硅转接板，PDF 给出的转接板面积约 `1200 mm²`（pp.328-329）。
3. `[PDF]` TSMC `CoWoS Roadmap`：从 `1× reticle` 向更大 interposer 扩展，用于容纳更大处理器和更多 HBM（pp.329-330）。

## 3. System-on-Wafer 起源型拓扑

```txt
多种功能器件：ASIC / memory / PMIC / MEMS / passives
→ 器件并排安装在带 TSV 和 RDL 的硅晶圆或硅转接板上
→ interposer 内部 RDL 负责横向系统互连
→ TSV / solder bump / solder ball
→ package substrate 或 PCB
```

Leti 的 SoW 更像 2.5D 的早期系统级原型：它强调用硅转接板承载多种异质器件，而不是只服务于 GPU + HBM。

![Leti SoW](/images/2.5D/Leti-SoW.png)

图片来源：`[PDF]` p.300，Fig. 6.1。该 SoW 被 PDF 作为 2.5D IC 集成的早期起源案例。

PDF 收录方案：

1. `[PDF]` Leti `SoW`（system-on-wafer）：ASIC、memory、PMIC、MEMS 与无源器件集成在带 TSV/RDL 的硅晶圆上；切割后形成 system-in-package 单元（pp.299-300）。
2. `[PDF]` IME `Double-Stacked Passive TSV Interposers`：两层 passive TSV interposer 承载 RF、logic 和堆叠 memory，属于多层硅载体的研发验证方案（pp.305-306）。

## 4. 双面芯片贴装的 2.5D 变体

```txt
heat spreader / heat sink
→ 顶面高功耗 flip-chip die，背面朝上，有源面朝下
→ microbumps + underfill
→ passive TSV interposer with RDL
→ 底面低功耗 chip / memory，连接到 interposer 背面
→ C4 bumps + underfill
→ organic package substrate
→ BGA solder balls
→ PCB
```

双面贴装仍然以 passive TSV/RDL interposer 为核心，但把部分低功耗芯片或存储器放到 interposer 背面，用 TSV 连接上下两侧。

![TSV interposer 双面芯片贴装](/images/2.5D/TSV-interposer-double-sided.png)

图片来源：`[PDF]` p.308，Fig. 6.12。

PDF 收录方案：

1. `[PDF]` HKUST `TSV Interposer with Chips on Both Sides`：高功率芯片在顶面，低功率芯片在底面，并配合 heat spreader（p.308）。
2. `[PDF]` ITRI `Thermal Management with Chips on Both Sides`：双面芯片 + Cu heat slug 的热管理方案（pp.308-311）。
3. `[PDF]` ITRI `Double-Sided Chip Attachments`：顶面 CPU/logic，底面双 memory 的双面贴装方案（pp.315-317）。
4. `[PDF]` ITRI `TSV Interposer with Chips on Both Sides`：顶面两颗芯片、底面一颗芯片的双面结构（pp.317-318）。

## 5. 半嵌入和 TSH 低成本变体

### 5.1 半嵌入 TSV/RDL/IPD interposer

```txt
塑封料 / 可选 heat spreader
→ 多颗 silicon die，正面朝下
→ microbumps + underfill
→ TSV/RDL/IPD interposer 的上半部
→ interposer 半嵌入 organic substrate / PCB 的腔体中
→ bumps + underfill
→ organic substrate / PCB
```

**与英特尔 EMIB 的异同：**
虽然概念上都属于“桥接”技术，但 ITRI 强调的 Semi-Embedded TSV Interposer 往往在桥接块内部保留了 TSV。

![半嵌入 TSV interposer](/images/2.5D/semi-embedded-TSV-interposer.png)

图片来源：`[PDF]` p.315，Fig. 6.21。

PDF 收录方案：

1. `[PDF]` ITRI `Semi-Embedded TSV Interposer`：转接板半嵌入有机基板，并保留 stress-relief gap；PDF 将其列为低轮廓、短设计周期、可返修的 2.5D 变体（p.315）。

### 5.2 TSH interposer

```txt
塑封料
→ 顶面 chip 与底面 chip
→ micro solder joints
→ through-silicon hole interposer（硅孔不做侧壁金属化，也不填铜）
→ Cu wire / Cu pillar 穿过硅孔连接上下芯片
→ interposer 表面 RDL 连接同侧芯片
→ organic package substrate
→ BGA solder balls
→ PCB
```

TSH（through-silicon hole）不再是严格的 Cu-filled TSV interposer，因为硅孔不做金属化和填铜；但它仍使用硅载体与 RDL 来组织上下/横向互连，因此 PDF 将其作为 2.5D 章节中的低成本 interposer 变体。

![TSH interposer](/images/2.5D/TSH-interposer.png)

图片来源：`[PDF]` p.321，Fig. 6.27。

PDF 收录方案：

1. `[PDF]` ITRI `TSH Interposer`（through-silicon-hole interposer）：通过无金属化硅孔、Cu pillar 和微焊点连接上下芯片，同侧芯片则通过 interposer RDL 互连（pp.318-322）。
