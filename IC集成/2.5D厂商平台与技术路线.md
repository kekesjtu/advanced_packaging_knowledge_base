# 2.5D 厂商平台与技术路线

2.5D 厂商平台的共同目标是：在同一封装内横向集成大逻辑芯片与 HBM / chiplets，用比封装基板更细的互连层实现高带宽、低延迟互连。

严格按本仓库的分类，典型 2.5D 平台是 `logic / HBM → microbumps → passive silicon interposer（含 TSV）→ package substrate`。RDL interposer、embedded bridge 和 organic interposer 虽然常被厂商放在同一先进封装产品族里，但更接近 2.3D 或 2.1D/2.3D 边界路线。

## 1. TSMC CoWoS-S：硅 TSV 转接板

```txt
heat spreader / mold
→ logic chiplets + HBM stacks，并排且有源面朝下
→ microbumps + underfill
→ silicon interposer 顶部高密度 RDL
→ silicon interposer TSV
→ C4 bumps
→ package substrate
→ BGA solder balls
→ PCB
```

CoWoS-S 是最符合本仓库 2.5D 定义的 TSMC 路线：核心载体是 silicon interposer，既提供细间距横向 RDL，也通过 TSV 把信号、电源和地引到封装基板。

![TSMC CoWoS-S](https://3dfabric.tsmc.com/site_img/dedicatedFoundry/technology/cowos/CoWoS-s_01.png)

图片来源：`TSMC 官方` CoWoS 页面，CoWoS-S section。

![TSMC CoWoS + DTC](/images/2.5D/CoWoS-DTC.png)

图片来源：`[PDF]` p.331，Fig. 6.41。

说明：

1. `TSMC 官方` `CoWoS-S`：基于 silicon interposer 的 chip-on-wafer-on-substrate，面向 AI 和 supercomputing，可承载 logic chiplets 与 HBM。
2. `TSMC 官方` CoWoS-S 支持在 silicon interposer 中加入 deep trench capacitors（DTC），用于高性能系统的电源完整性。
3. `TSMC 官方` CoWoS-S 已扩展到 `3.3× reticle` interposer，官网给出的面积约 `2700 mm²`。
4. `[PDF]` TSMC `CoWoS + DTC`：书中收录 logic + HBM2E + DTC 的 CoWoS 结构，说明 DTC 作为 interposer 内部电容用于供电网络支持（pp.330-331）。

## 2. TSMC CoWoS-R / CoWoS-L

### 2.1 CoWoS-R

```txt
heat spreader / mold
→ logic chiplets + HBM stacks
→ microbumps + underfill
→ RDL interpose
→ package substrate
→ BGA solder balls
→ PCB
```

CoWoS-R 仍属于 TSMC CoWoS 家族，但核心载体从 silicon interposer 变成 RDL interposer。按本仓库分类，它更接近 2.3D 的 RDL 转接板路线。

![TSMC CoWoS-R](https://3dfabric.tsmc.com/site_img/dedicatedFoundry/technology/cowos/CoWoS-R_05.png)

图片来源：`TSMC 官方` CoWoS 页面，CoWoS-R section。

### 2.2 CoWoS-L

```txt
heat spreader / mold
→ logic chiplets + HBM stacks
→ microbumps + underfill
→ RDL-based interposer
→ embedded local silicon interconnect
→ package substrate
→ BGA solder balls
→ PCB
```

CoWoS-L 结合 RDL-based interposer 与 embedded local silicon interconnect。它保留高密度局部互连能力，但不再是整片 passive TSV silicon interposer，因此更接近 2.1D local bridge 与 2.3D RDL interposer 的组合。

![TSMC CoWoS-L](https://3dfabric.tsmc.com/site_img/dedicatedFoundry/technology/cowos/CoWoS-L_01.png)

图片来源：`TSMC 官方` CoWoS 页面，CoWoS-L section。

官方资料：`https://3dfabric.tsmc.com/english/dedicatedFoundry/technology/cowos.htm`

## 3. Samsung 2.5D Cube-S：硅转接板

```txt
heat spreader / mold
→ logic chips + HBM dies / HBM modules，并排且有源面朝下
→ microbumps + underfill
→ silicon interposer 顶部 RDL
→ silicon interposer with TSV
→ package substrate
→ BGA solder balls
→ PCB
```

Cube-S 是 Samsung 官方先进异构集成页面中最接近严格 2.5D 定义的路线。它使用 silicon interposer 横向连接 logic 与 HBM。

![Samsung 2.5D Cube-S](/images/2.5D/Samsung-advanced-package-f1.png)

图片来源：`Samsung 官方` Advanced Heterogeneous Integration 页面，2.5D Cube-S section。

说明：

1. `Samsung 官方` `2.5D Cube-S`：使用 silicon interposer，面向 logic chips 与 HBM dies 的横向异构集成。
2. `Samsung 官方` Cube-S 强调 high bandwidth 和 low latency data path，应用目标包括 hyperscale data center、AI、HPC、network 和 5G。
3. `Samsung 官方` 页面说明 `3.3× silicon interposer`、advanced logic 与最多 8 个 HBM module 的 2.5D package 已完成 qualification 并可用于 production。

## 4. Samsung Cube-E / Cube-R

### 4.1 Cube-E

```txt
heat spreader / mold
→ logic chips + HBM
→ microbumps + underfill
→ silicon bridge + TSV-less RDL interposer
→ FO-PLP / package substrate
→ BGA solder balls
→ PCB
```

Cube-E 使用 silicon bridge 与 TSV-less RDL interposer，并结合 FO-PLP。它不使用完整大面积 TSV silicon interposer ，按照本仓库定义，它更接近 2.1D embedded bridge 与 2.3D fan-out/RDL interposer 的组合。

![Samsung Cube-E](/images/2.5D/Samsung-advanced-package-f2.png)

图片来源：`Samsung 官方` Advanced Heterogeneous Integration 页面，Cube-E section。

### 4.2 Cube-R

```txt
heat spreader / mold
→ logic chips + HBM
→ microbumps + underfill
→ organic RDL interposer
→ package substrate
→ BGA solder balls
→ PCB
```

Cube-R 使用 organic RDL interposer，因此更适合归到 2.3D 的 organic / RDL interposer 路线中。

![Samsung Cube-R](/images/2.5D/Samsung-advanced-package-f3.png)

图片来源：`Samsung 官方` Advanced Heterogeneous Integration 页面，Cube-R section。

官方资料：`https://semiconductor.samsung.com/foundry/advanced-package/advanced-heterogeneous-integration/`

## 5. 对照关系

1. 严格 2.5D：TSMC `CoWoS-S`、Samsung `Cube-S`。共同点是 silicon interposer、HBM/logic 并排、interposer RDL 横向互连、TSV 垂直引出。
2. 2.3D / 2.1D 组合：TSMC `CoWoS-R`、Samsung `Cube-R` 更偏 RDL interposer；TSMC `CoWoS-L`、Samsung `Cube-E` 引入局部 silicon bridge。
