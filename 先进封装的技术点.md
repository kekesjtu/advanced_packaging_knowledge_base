## 3. 晶圆/面板级封装 WLP/PLP

### 3.1 Fan-in WLP

Fan-in WLP 在原始晶圆上完成钝化层、RDL、UBM 和焊球。

所有的重布线层（RDL）和锡球（Bumps）都排布在芯片原本的尺寸范围内。

封装后的芯片尺寸和裸芯片（Die）几乎完全一样，因此它属于典型的WLCSP（晶圆级芯片尺寸封装）。

相比于Flip-Chip，它省去独立封装基板，封装尺寸接近裸片，适合尺寸较小、I/O 数量有限的器件。

### 3.2 Fan-out WLP

当芯片太小、管脚太多时，Fan-in装不下了。于是，工程师先将芯片切成单颗，再像拼图一样，把芯片重新组装到一个更大的临时基板上，用环氧树脂（EMC）灌封，形成重构晶圆。

这样，芯片四周就多出了一圈“人造空地”，重布线层可以延伸（扇出）到芯片外部。

WLP 使用圆形晶圆设备，工艺生态成熟；PLP 提高面积利用率和潜在产能，但对大尺寸面板的翘曲、设备兼容性和工艺均匀性要求更高。

### 3.3 Chip-first 与 Chip-last

Chip-first 先放置芯片并重构载体，再在芯片焊盘和 EMC 上直接制作 RDL，互连路径短且通常没有独立微凸点界面，但已知良品芯片（KGD）可能经历全部 RDL 制程，且 die shift 会直接消耗光刻对位裕量。

Chip-last 又称 RDL-first，先制作并可测试 RDL 载体，再贴装芯片。它有利于筛除不良 RDL、保护芯片免受前段制程影响，但增加微凸点/键合界面，并要求控制贴装共面性和底填。

> 注：KGD经历的后续工艺越少，最终良率才会越高。

## 4. 重布线层（RDL）

### 4.1 RDL 结构与功能

RDL 通过“介质层 + 铜走线 + 铜过孔”的重复堆叠，把芯片上细间距焊盘重新分配到凸点、焊球或相邻芯片。它承担三类功能：

- I/O 扇入或扇出。
- 芯片间高密度横向互连。
- 电源/地分配、阻抗控制和封装级无源结构。

主要设计参数是线宽/线距、铜厚、过孔直径与间距、层数、焊盘/凸点间距、走线长度、电流密度和介质厚度。缩小线宽/线距可提高布线密度，但会收紧光刻、对位、残铜、开路/短路和电迁移的工艺窗口。

典型加成法流程为：

```txt
介质涂布、固化与开孔
→ 溅射阻挡层/铜种子层
→ 光刻形成电镀图形
→ 铜电镀
→ 去胶
→ 蚀刻暴露的种子层
→ 重复堆叠并形成终端焊盘
```

### 4.2 导体与介质材料

| RDL 类型   | 常见介质                      | 特点与适用位置                                                          |
| ---------- | ----------------------------- | ----------------------------------------------------------------------- |
| 有机 RDL   | PI、PBO、BCB 等聚合物         | 工艺温度相对较低，适合扇出和封装基板；需控制吸湿、固化收缩和 CTE 不匹配 |
| 无机 RDL   | SiO2、SiNx 等                 | 尺寸稳定，可支持更细线宽/线距；设备和成本接近晶圆制造                   |
| Hybrid RDL | 芯片侧无机细线 + 外围有机粗线 | 兼顾超细间距和较大扇出面积，但增加异质界面和制程复杂度                  |

导体通常为铜，并配合 Ti、TiW、Ta/TaN 等粘附或阻挡层。高速链路除几何尺寸外，还要关注介质的介电常数 `Dk`、损耗因子 `Df`、铜面粗糙度和过孔回流路径。

## 5. TSV

### 5.1 TSV 类型与应用

TSV 是贯穿硅衬底的垂直导电通道。在 [2.5D](2.5D-IC集成.md) 中，TSV 位于无源硅转接板内，负责把顶部 RDL 的信号和电源引到底部封装基板；在 [3D](3D-IC集成.md) 中，TSV 可位于逻辑、存储器或有源基底芯片中，实现芯片堆叠的垂直互连。

按填充方式可分为实心铜 TSV、环形 TSV 等；按结构可分为盲孔制程后减薄露头和直接通孔。几何设计需同时考虑直径、深宽比、间距、寄生参数、载流能力和硅中应力影响。

### 5.2 Via-first、Via-middle 与 Via-last

| 集成时机   | 制作阶段                     | 主要特点                                                                   |
| ---------- | ---------------------------- | -------------------------------------------------------------------------- |
| Via-first  | 晶体管制造之前               | 可承受较高后续温度，但 TSV 会占用器件布局区域并受前段洁净要求约束          |
| Via-middle | 晶体管完成后、顶层互连完成前 | 兼顾互连接近性和制程整合，需限制热预算                                     |
| Via-last   | 晶圆互连完成后               | 与既有芯片流程解耦；可从正面或背面加工，但深宽比、对准和薄晶圆处理更具挑战 |

选择不只取决于成本，还取决于代工流程所有权、允许温度、TSV 与有源区距离以及是否需要背面供电或背面 RDL。

### 5.3 深硅刻蚀、绝缘、填充、减薄与露头

典型铜 TSV 流程为：

```txt
光刻定义 TSV
→ DRIE 深硅刻蚀
→ 侧壁绝缘层
→ 阻挡层和铜种子层
→ 铜电镀填充
→ CMP 去除表面过镀铜
→ 临时键合到载片
→ 背面减薄
→ TSV 露头、钝化和背面 RDL / UBM
→ 解键合
```

关键风险包括 DRIE 侧壁扇贝和底部缺口、绝缘层针孔、种子层断裂、填铜空洞、CMP 凹陷、减薄破片、铜泵出，以及铜/硅 CTE 不匹配形成的应力和器件禁布区。薄晶圆在后续键合、清洗和搬运中必须依赖稳定的临时载片系统。

## 7. 封装基板、桥接结构与转接板

### 7.1 积层封装基板

积层基板通常由芯层、两侧 build-up 介质/铜层、激光微孔和通孔组成，负责芯片扇出、电源分配和连接 PCB。设计重点是层叠对称、微孔可靠性、铜密度、阻抗控制、低损耗材料和封装翘曲。

### 7.2 无核心有机基板

无核心基板取消厚芯层，以更薄的积层结构实现较短互连和较小微孔。其主要挑战是制造和组装过程刚度不足、尺寸稳定性较差及翘曲较大，通常需要临时载体和对称布线。

### 7.3 嵌入式桥接结构

桥只覆盖相邻芯片之间需要高密度连接的局部区域，通常由硅或高密度薄膜结构制成。高速 die-to-die 信号经桥传输，电源和低密度 I/O 仍进入有机基板或扇出 RDL。与整片硅转接板相比，它降低硅面积和 TSV 需求，但增加桥的嵌入、对准和跨材料过渡设计。

### 7.4 有机与扇出 RDL 转接板

这类转接板覆盖封装主要区域但不含 TSV，通过有机积层或重构载体上的多层 RDL 实现横向互连，再以 C4 等连接下方基板。它们对应 [2.3D IC 集成](2.3D-IC集成.md)，成本和大尺寸能力通常优于硅转接板，但在线宽/线距、尺寸稳定性和散热方面存在不同约束。

### 7.5 无源/有源 TSV 硅转接板

无源硅转接板只含 RDL、TSV 和可选无源器件，对应 [2.5D](2.5D-IC集成.md)；有源硅基底还集成逻辑、缓存、路由或电源功能，对应 [3D](3D-IC集成.md)。硅平台可提供高密度、低尺寸偏差的互连，但成本、良率、大面积硅利用率和 TSV 工艺是主要约束。

### 7.6 载体选型

| 需求                          | 优先考虑                    |
| ----------------------------- | --------------------------- |
| 低成本、中等 I/O 密度         | 积层有机基板                |
| 仅局部需要超高密度 die-to-die | 嵌入式桥                    |
| 大面积、无 TSV 的高密度互连   | 无核心有机或扇出 RDL 转接板 |
| 全区域超高密度互连、HBM 接入  | 无源 TSV 硅转接板           |
| 需要基底逻辑、缓存或主动路由  | 有源 TSV 基底芯片           |

## 8. 材料与界面

### 8.1 EMC、底填料与模塑底填

EMC 为芯片和细线提供机械保护，也参与封装翘曲和热应力平衡。材料选择要关注玻璃化转变温度、CTE、弹性模量、固化收缩、吸湿率、填料尺寸和流动性。微凸点间隙越小，对底填填料粒径、黏度、润湿和无空洞填充的要求越高。

### 8.2 聚合物介质与钝化层

PI、PBO、BCB 和干膜介质用于 RDL 绝缘、缓冲和钝化。高速应用关注低 `Dk`、低 `Df` 和低吸湿；可靠性关注铜附着、固化程度、裂纹、离子污染和多次回流后的稳定性。低 CTE 有助于减小尺寸漂移和界面应力，但也需与模量、韧性和加工性一起权衡。

### 8.3 焊料、铜与表面处理

焊料连接依靠润湿和金属间化合物形成接头；必须控制焊料成分、回流峰值/时间、助焊剂残留和界面脆化。铜互连需关注氧化、电迁移和热机械疲劳。常见终端表面处理用于保护铜、提供可焊或可键合表面，选择时应匹配焊接、引线键合或混合键合要求。

### 8.4 临时键合与解键合材料

薄晶圆、重构晶圆和 RDL-first 载体常通过胶层临时固定在玻璃或硅载片上。材料需要承受后续温度、真空、湿法化学品和机械载荷，并在激光、热或机械解键合时低残留、低应力。粘结不足会造成渗液和滑移，粘结过强则可能在解键合时破片或拉裂 RDL。
