# 先进封装中的凸点与焊点（Bumps and Joints）

## 1. 专题范围

本页讲解先进封装中常见凸点、焊球和键合接点的：

- **结构**：凸点由哪些金属层组成，位于哪两个对象之间。
- **作用**：承担电气、机械、热传导或间隙控制中的哪些功能。
- **适用范围**：主要用于哪一级互连以及哪些封装类型。
- **区别**：C4、C2、微凸点、BGA 焊球和无凸点键合之间如何区分。

本文中的“凸点”主要指组装前形成在芯片、晶圆、转接板或封装上的凸起端子；“焊点/接点”指组装后，上下两侧端子完成接合形成的完整互连。

## 2. 区分凸点、焊点、焊盘和 UBM

### 2.1 组装前的凸点

```txt
芯片有源层
→ 芯片顶层焊盘
→ UBM
→ 焊料凸点（C4），或铜柱 + 焊料帽（C2）
```

此时凸点只位于连接的一侧，还没有焊，没有形成完整焊点。

### 2.2 组装后的焊点

```txt
芯片焊盘
→ UBM
→ 凸点主体
→ 接合界面 / 金属间化合物（IMC）
→ 载体侧表面处理与焊盘
→ RDL、转接板或封装基板
```

组装（焊）完后，凸点与对侧焊盘形成电气和机械连接，才构成完整焊点。

### 2.3 UBM 是什么

UBM（Under-Bump Metallization，凸点下金属层）是位于芯片焊盘与凸点之间的一层或多层薄膜金属结构，不是凸点本体。典型 UBM 包含以下功能层：

| 功能层      | 主要作用                                 |
| ----------- | ---------------------------------------- |
| 粘附层      | 让 UBM 与钝化层、铝焊盘或铜焊盘可靠结合  |
| 扩散阻挡层  | 抑制焊料、铜和芯片焊盘之间过度扩散       |
| 导电/种子层 | 为电镀铜柱或焊料提供连续导电路径         |
| 可润湿层    | 让熔融焊料稳定润湿并形成接头             |
| 防氧化层    | 保护铜或镍表面，保证储存和组装后的可焊性 |

没有合适的 UBM，凸点可能出现不润湿、界面脆化、焊盘侵蚀或扩散失效。

## 3. 凸点的分类维度

凸点名称来自不同分类维度，不能简单视为互斥类别。

| 分类维度     | 典型术语                                 | 说明                                       |
| ------------ | ---------------------------------------- | ------------------------------------------ |
| 按主体结构   | C4、C2、铜柱、金凸点、铜核焊球           | 描述凸点由什么构成                         |
| 按尺寸和应用 | Microbump（微凸点）                      | 描述细间距、小尺寸应用，不限定唯一材料结构 |
| 按互连层级   | 芯片凸点、转接板凸点、BGA/WLCSP 焊球     | 描述连接位于芯片级还是封装外部             |
| 按接合材料   | SAC、SnAg、SnCu、In、AuSn、Cu-Sn         | 描述焊料或金属体系                         |
| 按接合机理   | 回流焊、TCB、固液互扩散、Cu-Cu、混合键合 | 描述怎样形成最终接点                       |

## 4. 常见凸点与焊球

### 4.1 C4 焊料凸点

C4 是 Controlled Collapse Chip Connection 的缩写。其主体主要由焊料构成，回流时焊料熔化、润湿并塌陷到由焊料体积、焊盘尺寸和表面张力决定的高度。

**典型结构**

```txt
芯片焊盘
→ UBM
→ 大体积焊料凸点
→ 回流形成的焊料接头
→ 基板或转接板焊盘
```

![alt text](/pictures/bumps/C4与C2.png)

**主要作用**

- 把倒装芯片连接到陶瓷、硅或有机封装基板。
- 把硅转接板连接到下方封装基板。
- 同时提供电气连接、机械支撑和一定的热传导路径。
- 利用较多焊料和回流表面张力获得一定自对准能力。

**主要适用封装**

- [2D 倒装芯片](IC集成/2D-IC集成.md#2-倒装芯片flip-chip)。
- [2.1D](IC集成/2.1D-IC集成.md) 和 [2.3D](IC集成/2.3D-IC集成.md) 中相对较粗间距的芯片或载体连接。
- [2.5D](IC集成/2.5D-IC集成.md) 中硅转接板到底部封装基板的连接。
- [3D](IC集成/3D-IC集成.md) 中底层芯片或有源基底到封装基板的连接。

总结来说，往往用于芯片与基板，或转接板与基板的连接。

**与其他凸点的核心区别**

C4 以焊料为主体，焊料体积相对较大，接合时会明显塌陷。它的工艺成熟、应变顺应性较好，但继续缩小间距时容易受到桥连、焊料塌陷和焊点高度控制的限制。

### 4.2 C2 铜柱焊料帽凸点

C2 通常指 Chip Connection，其典型结构是铜柱（Cu pillar）顶部带一层焊料帽（solder cap）。铜柱是主要支撑和导电主体，焊料帽用于与对侧焊盘形成冶金接合。

**典型结构**

![alt text](/pictures/bumps/C2具体结构.png)

**主要作用**

- 通过铜柱保持稳定的互连高度，限制焊料回流时的塌陷。
- 减少焊料横向扩展，支持比传统 C4 更细的间距。
- 利用铜的低电阻和高导热性改善电气、供电和散热性能。
- 为 TCB、NCF/NCP 和细间距倒装芯片提供适合的端子结构。

**主要适用封装**

- [2D Chip-last Fan-out](IC集成/2D-IC集成.md#5-chip-last-扇出集成) 中芯片与预制 RDL 的连接。
- [2.1D](IC集成/2.1D-IC集成.md) 中芯片与顶部薄膜层或桥接区的连接。
- [2.3D](IC集成/2.3D-IC集成.md) 中芯片与无核心有机/RDL 转接板的连接。
- [2.5D](IC集成/2.5D-IC集成.md) 中芯片与硅转接板的细间距连接。
- [3D](IC集成/3D-IC集成.md) 中芯片堆叠和 HBM 层间连接。

总结来说，往往用于芯片与基板/RDL/转接板的连接，也可用于芯片之间连接。

**与 C4 的核心区别**

| 对比项       | C4                 | C2                           |
| ------------ | ------------------ | ---------------------------- |
| 主体         | 大部分为焊料       | 大部分为铜柱，顶部为少量焊料 |
| 回流塌陷     | 较明显             | 由铜柱限制                   |
| 自对准能力   | 相对较强           | 焊料量少，自对准能力较弱     |
| 间距缩小能力 | 相对有限           | 更适合细间距                 |
| 电阻与导热   | 主要受焊料影响     | 铜柱提供更低电阻和更高导热   |
| 高度控制     | 依赖焊料体积和润湿 | 主要由铜柱高度决定           |

### 4.3 微凸点（Microbump / µBump）

**名称与定义**

微凸点不是单一固定结构，而是用于细间距、高密度芯片互连的小尺寸凸点类别。它可以采用：

- 小型铜柱 + SnAg 等焊料帽（与C2相同，实际上就是把传统C2做小了）
- 铜/焊料/铜的夹层结构。
- 小体积焊料凸点。
- Cu-Sn 等固液互扩散结构。

最常见的微凸点外观与缩小后的 C2 相似，因此“C2”和“微凸点”经常重叠，但两者含义不同：`C2` 强调铜柱加焊料帽的结构，`微凸点` 强调尺寸、间距和高密度用途。

**典型结构**

```txt
上层芯片焊盘
→ UBM / 铜柱
→ 薄焊料层
→ 对侧铜焊盘或铜柱
→ 下层芯片、硅转接板或 RDL
```

**主要作用**

- 在极短距离内提供大量并行信号、电源和地连接。
- 连接 chiplet、HBM、硅转接板及垂直堆叠芯片。
- 缩短信号路径，降低互连电阻、电感和传输能耗。
- 支持比普通封装基板凸点更高的 I/O 密度。

**主要适用封装**

- [2.1D](IC集成/2.1D-IC集成.md) 局部桥和顶部细线层。
- [2.3D](IC集成/2.3D-IC集成.md) 高密度 RDL 转接板。
- [2.5D](IC集成/2.5D-IC集成.md) 芯片/HBM 到无源硅转接板。
- [3D HBM](IC集成/3D-IC集成.md#22-hbm-组装) 的 DRAM 层间连接。
- [基于 TSV 的 3D CoC](IC集成/3D-IC集成.md#23-基于-tsv-的-coc)。

总结来说，作用和C2类似，感觉上就是更新工艺的C2。C2用的回流焊，微凸点用TCB。

![alt text](/pictures/bumps/C4C2μbump对比.png)

**与传统C2/C4 的核心区别**

微凸点焊料体积更小、间距更细、单位面积数量更多。随着尺寸缩小：

- 焊料中的 IMC 比提高，接点可能逐渐接近全 IMC 结构。
- 自对准能力下降，对贴装精度和翘曲更敏感。
- 底填材料更难进入狭小间隙。
- 电流密度和热机械应力更集中。
- 批量回流窗口缩小，使用 TCB 而不是回流焊。

> 注：IMC介面金属共化物，焊锡（如锡膏中的纯锡 Sn）会与底层的金属（如铜 Cu）在高温下产生反应，生成一层IMC。它是连接焊料与基板的“桥梁”，代表焊接是否牢固。

### 4.4 铜柱或铜凸点（Cu Pillar / Cu Bump）

铜柱是以电镀铜形成的柱状凸起。“铜柱”是一个较宽泛的结构名称：

- 铜柱顶部带焊料帽时，通常属于 C2。
- 铜柱顶部采用镍、金或其他表面层时，可用于热压、热超声或导电胶连接。
- 铜直接与铜接合时，属于 Cu-Cu 键合，而不是传统焊料焊点。

**典型结构**

```txt
芯片焊盘
→ UBM
→ 铜柱
→ 焊料帽 / Ni-Au / Cu 接合表面
→ 对侧焊盘
```

**主要作用**

- 提供低电阻、高载流和较好的垂直导热路径。
- 精确控制芯片与载体之间的 stand-off 高度。
- 限制接合材料横向扩散，支持细间距。
- 作为 C2、微凸点和部分 Cu-Cu 键合的基础结构。

铜柱本身刚度较高，对热膨胀失配的顺应性低于大体积焊料，因此通常需要底填或合理的结构应力设计。

### 4.5 BGA ball 焊球

BGA（Ball Grid Array）焊球是位于封装底部的面阵列外部端子，用于把已经完成的封装连接到 PCB 或下一级封装。它属于封装级或板级互连，不是芯片有源面上的细间距凸点。

![alt text](/pictures/bumps/flip-chip中的BGA.png)

**主要作用**

- 把封装内部信号、电源和地引到 PCB。
- 承担封装重量并吸收封装与 PCB 之间的部分热机械变形。
- 为封装提供可进行 SMT 回流的标准化外部接口。

**主要适用封装**

- 2D、2.1D、2.3D、2.5D 和 3D 的基板型封装。
- PoP 上下封装连接。
- 大多数 BGA、FCBGA、SiP 和处理器封装。

总结来说，就是连接基板与PCB的。

### 4.6 WLCSP / Fan-out 焊球

**名称与定义**

WLCSP 或 Fan-out 焊球直接形成在芯片上的 RDL，或重构封装的 Fan-out RDL 上，而不是基板下面。它和 BGA 焊球都属于封装外部端子

**主要作用**

- 把芯片焊盘重新分配为可进行板级 SMT 的焊球阵列。
- 在不使用传统封装基板时直接连接 PCB。
- 降低封装厚度和互连长度。
- Fan-out 的焊球可以扩展到芯片投影之外，从而容纳更多 I/O。

实际上来说，也是一种 BGA ball，效果是很类似的。只不过一个连接的是基板和PCB，一个连接的是RDL和PCB。

## 5. 主要凸点总对比

| 类型               | 主要结构                | 所处层级                   | 核心作用                       | 相对间距能力 | 主要限制                    |
| ------------------ | ----------------------- | -------------------------- | ------------------------------ | ------------ | --------------------------- |
| C4                 | UBM + 大体积焊料        | 芯片到基板、转接板到基板   | 成熟倒装连接和应变顺应         | 较粗         | 塌陷、桥连，难以持续缩小    |
| C2                 | UBM + 铜柱 + 焊料帽     | 芯片到基板/RDL/转接板      | 控制高度、降低电阻、支持细间距 | 中到细       | 自对准较弱、铜柱刚度较高    |
| 微凸点             | 主要是小型铜柱+焊料     | 芯片到芯片、芯片到硅转接板 | 高密度、短距离互连             | 细           | 对位、翘曲、底填和 IMC 敏感 |
| 铜柱/铜凸点        | UBM + 铜柱 + 可选表面层 | 多种芯片级互连             | 高载流、导热和高度控制         | 中到细       | 顺应性低，需要应力管理      |
| BGA 焊球           | 封装焊盘 + 大焊球       | 封装到 PCB                 | 标准板级接口和机械顺应         | 较粗         | 封装/PCB 翘曲和板级疲劳     |
| WLCSP/Fan-out 焊球 | RDL + UBM + 焊球        | 无基板封装到 PCB           | 薄型直接板级连接               | 中           | 芯片/PCB CTE 失配           |

## 6. 凸点的共同作用

无论采用哪一种结构，凸点通常承担以下功能中的一项或多项：

### 6.1 电气连接

- 传输高速信号、低速控制信号、电源和地。
- 通过面阵列分布提高 I/O 数量。
- 通过多颗并联电源/地凸点降低供电阻抗。

### 6.2 机械支撑

- 保持芯片与载体之间的间隙。
- 支撑芯片重量并承受组装和使用载荷。
- 与底填共同分担温度循环、跌落和弯曲应力。

### 6.3 热传导

- 通过铜柱、焊料和电源/地凸点把部分热量传向载体。
- 为高功率芯片提供向下散热的并行路径。

凸点不是主要散热结构时，仍会影响芯片热点附近的垂直热阻。

### 6.4 间距转换

凸点与 RDL 配合，把芯片上的细间距焊盘连接到更粗间距的封装基板或 PCB。2.5D 和 3D 封装会采用多级凸点逐步完成间距放大。

## 7. 容易混淆的术语

### 7.1 C2 不等于所有微凸点

C2 是“铜柱 + 焊料帽”的结构；微凸点是细间距、小尺寸用途。小型 C2 可以是微凸点，但微凸点也可以采用其他金属结构。

### 7.2 C4 不等于 BGA 焊球

两者都可能以焊料为主体，但 C4 通常位于芯片或转接板内部互连层级，BGA 位于封装外部、连接 PCB。

### 7.3 焊球不等于焊点

焊球是组装前的球形端子；完成回流并连接上下焊盘后形成的整体才是焊点。

### 7.4 混合键合不是微凸点

混合键合通过平坦介质表面和嵌入式铜焊盘直接连接，不保留传统凸起端子，因此属于无凸点互连。
